# Kventure
A kstore web
This is a KO proyect named after its a virtual store of k-pop merch, its based on the necesity of a bussines of wanting to expand their work and achive more customers based , on a colorfull seting that represent the impact of music, and their meaning of their songs. Kventure — K-pop Universe Store A K-pop merchandise web store built with HTML5, CSS3 and Vanilla JavaScript. Browse official products from the most popular groups organized by record label.

Navigation by label → group → products
Persistent shopping cart (localStorage)
Login system with active session (sessionStorage)
Category filters for albums, lightsticks, plushies and clothing
Dynamic images connected from a JavaScript database
Fully responsive design with CSS animations
Unique visual theme per group with representative colors
Labels included

HYBE - BTS, TXT, ENHYPEN, Seventeen SM Entertainment - NCT, EXO, aespa YG Entertainment - BLACKPINK, BIGBANG, WINNER, iKON, 2NE1, TREASURE, CL JYP Entertainment - TWICE, Stray Kids P Nation - MAMAMOO

HTML5
CSS3 (custom properties, gradients, animations, Flexbox, Grid)
Vanilla JavaScript (no frameworks)
localStorage / sessionStorage
Google Fonts (Playfair Display, DM Sans)
